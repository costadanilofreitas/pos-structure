To create the edeployPOS installer follow the steps:

PRE INSTALLATION
 - pyinstaller must be installed
 - to install run the command -> pip install pyinstaller=3.5

INSTALLATION
 - Execute generate_installer.bat file
 - The files "edeployPOSInstaller - linux.zip" and "edeployPOSInstaller - windows.zip" will be generated. These zips contains all necessary files to install
