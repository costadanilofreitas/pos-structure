To create the executable (edeployPOSInstaller.zip) follow the steps:

PRE INSTALLATION
 - pyinstaller must be installed
 - to install run the command -> pip install pyinstaller=3.5

INSTALLATION
 - Review configurations.txt file to make sure that core and src versions are correct
 - Execute generate_installer.bat file
 - A edeployPOSInstaller.zip file will be generated. This zip contains all necessary files to install
